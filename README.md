This repository contains an Excel file for analyzing and visualizing electric vehicle market data.

OBJECTIVE: The objective of this project is to analyze and visualize electric vehicle (EV) data from various counties in Washington State. The data used in this project includes information on electric vehicles such as make, model, year, electric range, base MSRP, and geographic location within Washington State. The project aims to provide insights into the distribution, performance, and adoption of electric vehicles in the region through a comprehensive Excel dashboard.
