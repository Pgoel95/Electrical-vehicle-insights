This repository contains an Excel file for analyzing and visualizing electric vehicle market data.

# OBJECTIVE
The objective of this project is to analyze and visualize electric vehicle (EV) data from various counties in Washington State. The data used in this project includes information on electric vehicles such as make, model, year, electric range, base MSRP, and geographic location within Washington State. By providing a detailed and interactive dashboard, this project aims to enhance the understanding of electric vehicle adoption and support efforts to promote sustainable transportation solutions.
